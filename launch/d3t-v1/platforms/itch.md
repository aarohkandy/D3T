# itch.io

- status: ready after assets
- url: https://itch.io/game/new
- accepts: HTML5 embed ZIP from `npm run portal:build`
- assets: `cover_630x500.png`, 3-5 screenshots, optional banner, optional YouTube trailer
- sdk: none
- metadata: title, short description, HTML5 project kind, strategy/board-game tags, inputs, accessibility notes, free pricing
- revenue: free or pay-what-you-want; default itch split adjustable
- blockers: cover export, screenshots, final visibility choice
- gotchas: keep tags honest, mark public when ready, use butler later if uploads become frequent

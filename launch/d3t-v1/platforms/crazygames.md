# CrazyGames

- status: blocked
- url: https://developer.crazygames.com/
- accepts: HTML5 ZIP from `npm run portal:build:crazygames`
- assets: cover thumbnail, short gameplay trailer, 16:9 screenshots, mobile-readable screenshots
- sdk: CrazyGames SDK v3 through the portal SDK adapter only
- metadata: title `D3T`, description from `metadata.yml`, controls, strategy/board-game tags, orientation both
- revenue: ad revenue share; do not accept optional exclusivity unless launch strategy changes
- blockers: SDK preview/QA, ad-call review, final thumbnail/trailer/screenshots
- gotchas: gameplay in one click, no App Store links, no third-party ads, PEGI 12 compatible, readable at DPR 1

## Boilerplate

D3T is a browser strategy game based on three-layer Grandfather Tic-Tac-Toe. Players fight across 729 tiny cells, but each move routes the opponent into a forced board, turning simple marks into long-range traps.

## Short pitch

D3T takes Tic-Tac-Toe from a solved childhood game into a tense browser strategy duel: three nested boards, forced replies, live challenges, and a one-click portal version for instant play.

## Key talking points

- Three nested board layers with 729 playable cells.
- Forced-target mechanic makes each move route the next player.
- Main web app supports account-based friend challenges.
- Portal build supports one-click guest pass-and-play.
- Built with Next.js, TypeScript, Supabase, Postgres, and Vercel.
